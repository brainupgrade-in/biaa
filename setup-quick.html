Building Intelligent AI Agents — Participant Setup

5-day hands-on workshop · labs run on each participant’s own laptop · Trainer: Rajesh Gheware
In short: each participant needs a normal laptop with Python and Jupyter, a one-time software install (a single command we provide), and a few free accounts. No cost, no GPU, no cloud.

💻 Each participant’s laptop

  • Windows, macOS or Linux · 8 GB+ RAM · ~10 GB free disk space
  • Python 3.12 and Jupyter (or VS Code) installed
  • A one-time install of the workshop’s Python libraries — a single copy-paste command the trainer provides
  • A free AI model: either install Ollama (runs on the laptop) or use a free Groq API key

🔑 Free accounts to create before the workshop

  • Groq — the AI model service (skip if using Ollama locally)
  • Serper — Google Search for the agents (Day 3)
  • Wolfram Alpha — calculations for the agents (Day 3)
All three have free tiers — the trainer will guide participants through getting the keys.

🌐 For your IT team — please allow these websites

  • pypi.org, files.pythonhosted.org — software install
  • api.groq.com, ollama.com, storage.googleapis.com — AI model & data
  • google.serper.dev, api.wolframalpha.com — Day-3 agent tools
Good to know: No GPU, no cloud, and no paid subscription required. The trainer provides all lab notebooks and a short setup walk-through at the start. A detailed technical setup guide is available for IT teams on request.
