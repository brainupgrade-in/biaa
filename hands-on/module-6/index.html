Day 3 · Module 6 — Hands-on Labs

Frameworks for Building AI Agents — build a real LangChain agent
12 labs | 6 Beginner · 3 Intermediate · 3 Advanced | ~5h 35m total | Experiential: concept → demo → practice
How these labs work. Each notebook follows the same rhythm — read the Concept, run the Demo, then complete Your Turn by replacing every ___ placeholder. Run the grader cell at the bottom: it prints [PASS] / [FAIL] / [TODO] and a final Score. Each lab writes to its own /tmp/biaa-lab-06-NN/ working dir. Stuck? Every lab has a full answer key under solutions/ — try first, then check.
Setup (almost none). Just Python 3.10+ — every graded step uses only the standard library. You build a real LangChain workflow (@tool, PromptTemplate, create_react_agent, AgentExecutor with max_iterations, memory, a LangGraph-style state graph, guardrails and tracing) through a tiny LangChain-shaped shim — same names and shapes as the real library — driven by a deterministic scripted model, so there are no API keys, no model downloads, no network. The calculator/compute tools use a small AST-based safe evaluator (never bare eval). Each Advanced lab ends with an optional, non-graded cell that runs the real LangChain (pip install langchain langchain-ollama + ollama run llama3.2:1b, or ChatGroq with a GROQ_API_KEY; Google Serper / Wolfram need their own keys); it skips cleanly if absent.
