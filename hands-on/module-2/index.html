⌂ Course outline

Day 1 · Module 2 — Hands-on Labs

Introduction to Deep Learning — build, train & inspect neural networks
12 labs | 6 Beginner · 3 Intermediate · 3 Advanced | ~6h 15m total | Experiential: concept → demo → practice
How these labs work. Each notebook follows the same rhythm — read the Concept, run the Demo to see it work, then complete Your Turn by replacing every ___ placeholder. Run the grader cell at the bottom: it prints [PASS] / [FAIL] / [TODO] and a final Score. Each lab writes to its own /tmp/biaa-lab-02-NN/ working dir. Stuck? Every lab has a full answer key under solutions/ — try first, then check.
Setup (once). Python 3.10+ with: pip install numpy scikit-learn matplotlib. Labs 2.1–2.6 use NumPy/matplotlib; 2.7–2.9 add scikit-learn. The Advanced labs 2.10–2.12 use Keras / TensorFlow (pip install tensorflow-cpu) and train on MNIST handwritten digits; if MNIST cannot download, the labs fall back automatically to the offline 8x8 digit set so they still run. In the managed lab sandbox everything is pre-installed.
