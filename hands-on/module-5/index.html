Day 3 · Module 5 — Hands-on Labs

What is Agentic AI? — build a ReAct agent loop from scratch
12 labs | 6 Beginner · 3 Intermediate · 3 Advanced | ~5h 30m total | Experiential: concept → demo → practice
How these labs work. Each notebook follows the same rhythm — read the Concept, run the Demo, then complete Your Turn by replacing every ___ placeholder. Run the grader cell at the bottom: it prints [PASS] / [FAIL] / [TODO] and a final Score. Each lab writes to its own /tmp/biaa-lab-05-NN/ working dir. Stuck? Every lab has a full answer key under solutions/ — try first, then check.
Setup (almost none). Just Python 3.10+ — every graded step uses only the standard library. You build a working ReAct-style agent from scratch (tools, the reason → act → observe loop, ReAct parsing, tool routing, memory and guardrails), driven by a deterministic rule-based “mock LLM” policy — no API keys, no model downloads, no network. The calculator tool uses a small AST-based safe evaluator (never bare eval). Each Advanced lab ends with an optional, non-graded cell that swaps the rule-based policy for a real LLM via LangChain (pip install langchain-ollama + ollama run llama3.2:1b, or ChatGroq with a GROQ_API_KEY); it skips cleanly if absent.
