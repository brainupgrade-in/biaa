One neuron, learning the OR gate

The whole “Your Turn” snippet is a single artificial neuron: two inputs flow in, get weighted and summed, pass through a sigmoid, and come out as a decision. Below, every part of the picture is keyed to the exact line of code that builds it.

Each part → its line of code

The colours above match the swatches below.

Inputs  x₁, x₂

The two features of one example — a row of X. For OR, each is 0 or 1.

z = X.dot(w) + b # the X part

Weights  w₁, w₂ & bias b

The neuron’s learnable parameters — one weight per input, plus a bias. This is the model.

w = rng.normal(size=X.shape[1]); b = 0.0

Σ  weighted sum (z)

Multiply each input by its weight, add them, add the bias. One number per example — the “logit”.

z = X.dot(w) + b

σ  sigmoid activation

Squashes z into a probability between 0 and 1 — the neuron’s smooth “firing strength”.

def sigmoid(z): return 1/(1+np.exp(-z)) p = sigmoid(z)

Output  p → ŷ

The probability p, then thresholded at 0.5 to give a hard 0/1 prediction.

(sigmoid(X.dot(w)+b) >= 0.5).astype(int)

Learning  the loop

3000 times: predict, measure error, nudge every weight downhill against the gradient.

w -= lr * X.T.dot(error)/len(y) b -= lr * error.mean()

How it learns — one epoch, ×3000

The loop body is just four moves, repeated until the weights settle.

01 · FORWARD

Predict

Weighted sum then sigmoid.
p = sigmoid(X.dot(w)+b)

02 · COMPARE

Measure error

How far off, per example.
error = p - y

03 · GRADIENT

Find the slope

Blame each weight for the error.
X.T.dot(error)/len(y)

04 · UPDATE

Step downhill

Nudge weights & bias against it.
w -= lr * grad

Why this matters: this one neuron is the atom of every deep network. Keras and PyTorch in the later modules just stack millions of these and run the same four-step loop. One honest limit — OR is linearly separable, so a single neuron solves it; XOR is not, and this exact neuron would fail it. That failure is why hidden layers were invented.
