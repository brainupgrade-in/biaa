⌂ Course outline

Day 1 · Module 1 — Hands-on Labs

Understanding AI & its Evolution — learn by building
12 labs | 6 Beginner · 3 Intermediate · 3 Advanced | ~5h 20m total | Experiential: concept → demo → practice
How these labs work. Each notebook follows the same rhythm — read the Concept, run the Demo to see it work, then complete Your Turn by replacing every ___ placeholder. Run the grader cell at the bottom: it prints [PASS] / [FAIL] / [TODO] and a final Score. Each lab writes to its own temp working dir (biaa-lab-01-NN/). Stuck? Every lab has a full answer key under solutions/ — try first, then check.
Setup (once). Python 3.10+ with: pip install numpy scikit-learn matplotlib. Labs 1.1–1.6 & 1.10 & 1.12 need only Python/NumPy; 1.7–1.9 & 1.11 use scikit-learn (1.11 also matplotlib). Lab 1.12 has an optional step using a local LLM via Ollama (pip install langchain-community) — safe to skip. In the managed lab sandbox everything is pre-installed.
