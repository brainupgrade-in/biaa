Day 2 · Module 3 — Hands-on Labs

Why Transformers? — tokenize, embed, attend & generate
12 labs | 6 Beginner · 3 Intermediate · 3 Advanced | ~6h 15m total | Experiential: concept → demo → practice
How these labs work. Each notebook follows the same rhythm — read the Concept, run the Demo, then complete Your Turn by replacing every ___ placeholder. Run the grader cell at the bottom: it prints [PASS] / [FAIL] / [TODO] and a final Score. Each lab writes to its own /tmp/biaa-lab-03-NN/ working dir. Stuck? Every lab has a full answer key under solutions/ — try first, then check.
Setup (once). Python 3.10+ with: pip install numpy scikit-learn matplotlib. Every graded step runs offline on NumPy / scikit-learn — no API keys, no model downloads. Each Advanced lab also ends with an optional, non-graded Hugging Face cell (pip install transformers torch) that runs a real tiny transformer (e.g. bert-tiny, tiny-gpt2) so you can see the production tool; it skips cleanly if transformers is not installed. In the managed lab sandbox everything is pre-installed.
