Day 2 · Module 4 — Hands-on Labs

Pre-trained Models & Fine-tuning — load, adapt & fine-tune
12 labs | 6 Beginner · 3 Intermediate · 3 Advanced | ~5h 35m total | Experiential: concept → demo → practice
How these labs work. Each notebook follows the same rhythm — read the Concept, run the Demo, then complete Your Turn by replacing every ___ placeholder. Run the grader cell at the bottom: it prints [PASS] / [FAIL] / [TODO] and a final Score. Each lab writes to its own /tmp/biaa-lab-04-NN/ working dir. Stuck? Every lab has a full answer key under solutions/ — try first, then check.
Setup (once). Python 3.10+ with: pip install numpy scikit-learn matplotlib. Every graded step runs offline on NumPy / scikit-learn — no API keys, no model downloads. “Fine-tuning” is taught as transfer learning (a frozen feature extractor + a trainable head) so it runs anywhere. Each Advanced lab also ends with an optional, non-graded Hugging Face cell (pip install transformers torch) that runs / fine-tunes a real tiny model (e.g. bert-tiny) — the managed-sandbox target; it skips cleanly if transformers is absent. In the managed lab sandbox everything is pre-installed.
