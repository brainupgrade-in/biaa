Course Overview

A practitioner-first program that takes participants from zero AI background to building and deploying intelligent, autonomous AI agents. The week opens with the foundations — how modern AI works and how deep neural networks learn — then moves to the transformer architecture that powers today's large language models, including using pre-trained models and fine-tuning them for real tasks. From there the course pivots to the heart of the program: agentic AI — designing autonomous agents with frameworks such as LangChain and AutoGPT, wiring them to external tools, APIs and databases, automating real work, and coordinating multiple agents for decision-making. The final day applies everything to industry-specific agents for finance, healthcare, customer service and cybersecurity, and closes with ethics and responsible AI. Roughly 50% concept, 50% hands-on; every module pairs with a Python lab that participants run themselves. The five days build in a clear arc — understand the models → harness transformers → build your first agent → make agents do real work → ship industry agents responsibly.

No AI background needed. The course assumes no prior knowledge of AI, deep learning, or machine learning. Basic Python helps but is not mandatory — concepts are introduced from first principles, and every lab provides scaffolded, fill-in-the-blank starter code so participants of all levels can keep pace.
Practical, build-first focus. The emphasis is on building working agents, not deriving the math. Heavy theory (backpropagation derivations, attention proofs, distributed-training internals) is summarized at an intuition level so the week stays focused on shipping intelligent agents.

Objective & Learning Outcomes

Equip participants to go from AI fundamentals to building intelligent, autonomous AI agents that automate tasks, make decisions, and integrate with real systems — ready to apply in their own domain. By the end of the course, participants will be able to:

  • Build AI models using deep learning and transformer-based architectures
  • Implement agentic AI pipelines using frameworks like LangChain and AutoGPT
  • Design AI-driven task automation and decision-making systems
  • Develop industry-specific AI agents for finance, healthcare & customer service
  • Integrate AI agents with external tools, APIs and databases
  • Use and fine-tune pre-trained models (GPT, BERT) for real-world tasks
  • Coordinate multi-agent systems for collaborative decision-making
  • Apply responsible-AI practices and debug agent failures

Capstone — Your Own Industry-Specific Intelligent Agent

By the end of the week, each participant (or small team) builds and demos an intelligent, autonomous AI agent for a domain they care about — finance, healthcare, customer service or beyond. The agent is built on the frameworks covered in the course (LangChain / AutoGPT), automates a real task and makes decisions, integrates external tools, APIs and databases, and where useful coordinates multiple specialist agents. It is wrapped in responsible-AI guardrails — handling for bias, privacy and hallucination — and is debuggable, with tracing and basic evaluation in place. Each participant presents what their agent does, how it integrates with real systems, what it deliberately won't do, and how they'll extend it next.
